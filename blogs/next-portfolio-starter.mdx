The [Next.js Portfolio Blog Starter Kit](https://vercel.com/templates/next.js/portfolio-starter-kit), designed and presented by the team at Next.js, is an app template designed for building a comprehensive portfolio site, complete with a blog. This template simplifies the creation of a professional online presence and also enhances it with a suite of modern web technologies and design practices.

### Key Features

1. **MDX and Markdown Support**: This feature allows you to write your content using MDX or Markdown, offering a flexible way to integrate dynamic elements within static content.
2. **Optimized for SEO**: With built-in tools like a sitemap generator, robots.txt, and JSON-LD schema, this starter kit ensures your portfolio ranks well on search engines.
3. **RSS Feed and Dynamic OG Images**: Keep your audience updated with new posts via RSS feeds and attract social media engagement with dynamically generated Open Graph images.
4. **Syntax Highlighting**: Code blocks within your articles look clean and are easy to read, thanks to integrated syntax highlighting.
5. **Tailwind v4**: Utilize the latest features of Tailwind CSS for a highly customizable and responsive design.
6. **Performance Insights**: The kit is optimized for speed and integrates seamlessly with Vercel Speed Insights and Web Analytics.
7. **Geist Font**: This modern sans-serif typeface adds a clean and professional look to your site.

### How to Get Started

#### One-Click Deploy

For those looking to get up and running easily, the one-click deploy option via Vercel allows you to set up your portfolio in minutes. Deploy directly from [Vercel's Portfolio Blog Starter page](https://vercel.com/templates/next.js/portfolio-starter-kit).

#### Clone and Deploy

If you prefer a more hands-on approach:

- Use `pnpm create next-app --example https://github.com/vercel/examples/tree/main/solutions/blog blog` to clone the repository.
- Start your development server with `pnpm dev`.
